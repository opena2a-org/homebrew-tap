# OpenA2A Homebrew Tap

Homebrew formulae for the [OpenA2A](https://opena2a.org) security platform.

## Installation

```bash
brew tap opena2a-org/tap
brew install opena2a
```

## Upgrade

```bash
brew upgrade opena2a
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `opena2a` | Security platform CLI for AI agents - scan, verify, and protect |

## Usage

```bash
opena2a --help
opena2a scan <package>
opena2a protect
opena2a guard
```

## Alternative Installation

```bash
npm install -g opena2a
```
